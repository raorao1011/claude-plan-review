# Claude Plans Reviewer

VSCode拡張機能で、Claude Codeの`~/.claude/plans`ディレクトリ内のプランファイルを簡単にレビュー・管理できます。

## 主要機能

- **サイドバーに一覧表示**: プランファイルをTreeView形式で表示
- **作成日時でソート**: 最新のプランから順に表示（"Today", "Yesterday", "2 days ago"形式）
- **検索・フィルタリング**: タイトルやファイル名で素早く検索
- **マークダウンプレビュー**: VSCode標準プレビューで即座に内容を確認
- **ファイルパスジャンプ**: プラン内の`/Users/...`パスをクリックで開く
- **プラン削除**: 不要なプランファイルを簡単に削除
- **自動更新**: ファイルの作成・変更・削除を自動検知

## 使い方

1. VSCodeのアクティビティバーに表示される「Claude Plans」アイコンをクリック
2. サイドバーにプラン一覧が表示されます
3. プランをクリックしてマークダウンプレビューを表示
4. プレビュー内のファイルパスをCmd/Ctrl+クリックでコードを開く
5. 検索アイコンで特定のプランを検索
6. 右クリックまたはゴミ箱アイコンでプランを削除

## 要件

- Visual Studio Code ^1.85.0

## インストール

1. このプロジェクトをclone
2. `npm install`で依存パッケージをインストール
3. F5キーで拡張機能開発ホストを起動

## 開発

- `npm run compile`: TypeScriptコンパイル
- `npm run watch`: ファイル変更を監視して自動コンパイル
- `npm run lint`: oxlintでコードチェック
- `npm run format`: oxlintで自動フォーマット
